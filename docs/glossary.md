# Agentic Loop Glossary

Stable Agentic Loop vocabulary is defined in
[`AGENTIC_LOOP.md`](../AGENTIC_LOOP.md#glossary). The installed process document
is the runtime source for downstream agents.

Target projects can keep their own domain glossary in `AGENTS.md`,
`CONTEXT.md`, `docs/context.md`, or another project handbook. Record
non-standard source document paths in `.agenticloop/project.md` frontmatter.
